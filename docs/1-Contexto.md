# Contexto do Projeto

## Problema

O financiamento de pesquisas científicas é essencial para o desenvolvimento científico e tecnológico. No entanto, muitos pesquisadores enfrentam dificuldades para obter recursos financeiros para desenvolver seus projetos.

Entre os principais desafios estão a escassez de investimentos, a burocracia para participação em editais e a dificuldade de acesso às informações sobre oportunidades de financiamento.

### Matriz CSD

| Certezas | Suposições | Dúvidas |
|--------|--------|--------|
| Pesquisas precisam de financiamento | Muitos pesquisadores não conhecem editais | Como os pesquisadores encontram financiamento? |
| Universidades dependem de recursos externos | O processo de inscrição é burocrático | Quais são os critérios de aprovação? |
| Existem órgãos de fomento | Pesquisadores iniciantes têm mais dificuldade | O financiamento disponível é suficiente? |

## Público-Alvo

Para compreender melhor o problema do financiamento de pesquisas, foi elaborado um Mapa de Stakeholders que identifica os principais envolvidos nesse processo.

### Mapa de Stakeholders

**Pessoas Fundamentais**
- Pesquisadores
- Estudantes universitários
- Pós-graduandos
- Professores pesquisadores

**Pessoas Importantes**
- Universidades
- Instituições de pesquisa
- Agências de fomento (CNPq, CAPES, FAPEMIG)
- Empresas que investem em inovação

**Pessoas Influenciadoras**
- Governo
- Ministério da Ciência e Tecnologia
- Comunidade científica
- Sociedade
